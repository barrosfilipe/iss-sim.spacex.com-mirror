## How to run

- Clone this repository `git clone git@github.com:barrosfilipe/iss-sim.spacex.com-mirror.git`
- Run cd `iss-sim.spacex.com-mirror`
- Run `npm install` or `yarn`
- Run `npm start` or `yarn start`
- Go to: http://localhost:8080
